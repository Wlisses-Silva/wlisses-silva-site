# Wlisses Silva — Site Pessoal

Site profissional de Wlisses Silva, Técnico em Segurança do Trabalho e Meio Ambiente.

## 🔗 Acesse
[wlisses-silva-site.vercel.app](https://wlisses-silva-site.vercel.app)

## 📌 Sobre
Portfolio pessoal com informações sobre formação, habilidades, experiência profissional e formulário de contato integrado ao [FormSubmit](https://formsubmit.co).

## 🛠️ Tecnologias
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)
- [FormSubmit](https://formsubmit.co) — envio de e-mails pelo formulário de contato
- [Font Awesome](https://fontawesome.com) — ícones
- [Google Fonts](https://fonts.google.com) — tipografia

## 📂 Estrutura
```
wlisses-silva-site/
├── index.html        # Página principal
├── style.css         # Estilos
├── script.js         # Interatividade
├── wlisses-foto.jpg  # Foto de perfil
└── hero-bg.png       # Imagem de fundo do hero
```

## 📬 Formulário de Contato
O formulário utiliza o serviço **FormSubmit** para enviar mensagens diretamente para o e-mail do proprietário, sem necessidade de backend ou servidor.
